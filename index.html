Introduction to Computer

Introduction

Computer

Computer stands for Commonly Operating Machine Particular Use Tread Education And Rasearch. It is an electronic device that perform tasks in th principle of IPO(Input Processing Unit).Now a days computer is being one of the essential things for our life and in our modern world every work related to every field is being done with the help of computer.It is used for money transaction, online shopping, to pay bill in online, movie making, entertainment sector, record keeping, news telecasting, to get online eduaction and so on.

Data

Data is an raw fact or figure which may be letters, numbers, words or any symbol. It doesnot gives complete meaning.

Information

Information is a output which we get after processing a data. It gives complete meaning.

Instruction

The basic command that tells what type of action shouls be taken by Computer is called Introduction.

Computer system

Computer system is the combination of both computer hardware and software.Hardware and Software works together to get meaningful information.

Program

The set of Instruction given to the computer to perform some task is called Program.

Software

The group of program that tell the Computer to perform some specific task is called Software.

Cache memory

Cache memory is the fastest memory located between primary memory and CPU.

Functions of Computer

Input

Data and instruction whatever entered through the help of input device is called input. we can give command into computer by typing,touching on device through input device like:Keyboard,Mouse,Touchscreen,Touchpad,Joystick,etc.

Process

Process meant to treat the data accoarding to instruction given by the users. It is done through processing device and i.e CPU(Central Processing Unit).

Output

The result which we obtain after processing through the help of output device os called Output. We get output in printed and unprinted form through different output devices like:Monitor,Speaker,Printer,Projector,Plotter,etc.

Storage

The place on the Computer where we stores data,information,files,etc either permanently or temporarily. We can transfer files whenever we want through storage device.

  • 1 second = 10,00,00,00,00,000 Pico second

Automatic

Computer needs data nad instrauction. After getting data and instruction it does it's task automatically.

Diligence

Computer is a diligent machine. It performs the same tasks repeatily without losing speed and accuracy.

Versatile

Computer is a versatile machine. It can perform variety of task and because of that reason computer is being used in many different field.

Storage

The place where we stores our data, informations, files and folder on the computre is called storage.It stores files temprorarily as well as permanently.

Processing Device

CPU

CPU refers to Central Processing Unit. It is a processing device that processes the data. It performs arithmetic and logical calculation. It is also known as Microprocessor. It has three parts.

Parts of CPU
CU(Control Unit)

CU refers to the Control Unit. It controls all the instructions which is given into the computer by users.It has four main functions and they are:Fetch, decode, Execute and Run. It gives command to do work on data accoarding to the user demand.

ALU(Arithmetical Logical Unit)

ALU refers to the Arithmetical Logical Unit. It perform arithmetic calculation such as: Addition, Subtraction, Multipication and Division and also perform logical calculation. It performs the actual calculation.

MU(Memory Unit)

Mu refers to the Memory Unit.It is the special temporary memory of CPU. It stores the data during procesing.It is also called register unit.

Output Device

Softcopy Output

Output device which donot give output on paper is calles softcopy output. It's output can't be save permanently for the future. Some examples of output devices which donot gives output on printed form are as follow:

Monitor

Monitor is a softcopy output device.It gives output on screen in visual form.It is also caled VDU(Visual Display Unit). LCD(Liquid Crystal Display) or LED(Light Emmiting Diode), CRT(Cathode Ray Tube) are the types of Monitor.

CRT monitor

CRT refers to the Cathode Ray Tube.It is very large in size and heavy weight.It was developed in old time so, it uses old mechanism and technology. It consumed large amount of electricity.It display low graphic and nowadays, it is very cheap in cost.

LCD or LED monitor

LCD refers to the Liquid Crystal Display and LED refers to the Light Emmiting Diode.It is smal in size and light in weight.It is developed in present time so, It uses modern technology. It consumed less amount of electricity.It display high graphic.Nowadays, it is very expensive in cost.

Projector

Projector is a softcopy output device.It projects its output on screen in the visual form.They dont have hardware.

Speaker

Speaker is a softcopy output device.It produced sound generated bt the computer.Its output cannot be stored anly can be listened.

Hard copy Output

Output device which gives output on printed form on paper is called Hard copy Output. Output given by hardcopy output can be saved permanently for the future. Some of Hard copy output devices are:

Printer

Printer is an hard copy output device. It displayed its data in a printed form on the paper.We cannot recorect its output. Impact and non-impact printer are two types of Printer.

Impact Printer

Impact printer uses old technology or mechanism.It is slow and noisy.It uses Metal pin, drum, hammer, ink ribbon and carbon paper to print document.Dot matrix printer, Daisy wheel printer and Drum printer are the types of Impact printer.It is hard copy output device.

Non-impact printer

Non-impact printer uses new technology or mechanism.It is very fast in performance. It uses heat and pressure to print on a document.It is also a hard copy output device. Ink jet printer and laser printer are the two types of non-impact printer.

Laser Printer

Laser printer uses laser technology to print on document.It prints a document by spraying ink by the heat nd pressure produced by Printer.It uses ink of carbon dust.

Ink jet Printer

Inkjet printer print on document by spraying the fine tiny drop of ink and produced the result. It is mostyly used in photo studio.

Plotter

Plotter is a hardcopy output device.It is unlike a printer.It uses small pen which produced result by dropping a ink on a paper through this procedure it produces the result.

Storage device

Primary Memory

Primary memory is a internal memory which stores data for processing into the CPU. It communicate directly to the CPU.It has two types and they are:

RAM(Random Access Memory)

RAM refers to the Random Access Memory.It is a internal volatile memory.It stores the data which are currently being used by the computer.The contents of Ram gets lost when the computer is turned off.It is read/write memory or working memory of the computer.

ROM(Read Only Memory)

ROM refers to the Read Only Memory.It is a pre programmable non-volatile memory of the computer. It keeps the information of all input and output devices.The information of ROM cannot be deleted and modified, only can be read.It contains Boot programs, POST(Power On Self Test) and BIOS(Basic Input Output System).

Secondary Memory

Secondary memory is a External memory which stores data and information permantly. They transfer data and files into the RAM to used. We can easily attached or detached Secondary memory from computer. Data can be read and write.

HDD(Hard Disk Drive)

HDD refers to the Hard Disk Drive.It is used as secondary storage device in computer.It can store large amount of data permanantly.Data can be read and write into it.Data can be hold for a long time.Data are stored in on or off form.

CD(Compact Disk)

CD refers to the Compact Disk.It is used a secondary memory.Data can be read and write only through laser light that is why it is also called as optical disk.it is also known as erasable disk.It is permanent memory and holds data for a long time.It has storage capacity of 650-700mb.Data cannot be modified easily like in hard disk.It is easily portable and data are stored in on or off form.

DVD(Digial Video Disk)

DVD refers to the Digital Video Disk.It is permanent secondary storage device.It is also called optical disk as data can only read and write through laser light.Mainly they are used to keep backup copy of data and programs.It can store 4.7-17GB of data.It stores data in on or off form.

BD(Blueray Drive)

BD refers to the Blue Ray drive.It is permanent secondary storage device.Data can be read and write only through blue-violet laser light.It is also called an optical disk.It is used to store high quality of video.It's capacity is 25-50GB.It also stores data in on or off form.

Pendrive

Pendrive is a permanent secondary storage device.It is EEPROM(Electrically Erasable Programmable Read Only Memory).Data can be transfered very easily .It is very portable due to it's small size.It is also called a Flash memory.It stores 1-512GB of data.

Magnetic disk

Magnetic Disk is most popular direct access storage device.It stores large volume of data The data of magnetic disk can be accessed from any location on a disk directly without accessing the data storage previous to it. It is a circular disk made of metal or plastic and coated with magnetic material.The stored information can be changed at any time by remagnetizing the surface with the new information.

Magnetic Tape

Magnetic tape is a secondary storage device that stores a large volume of data,information or programs permanentaly on the magnetic coated surface.It has high storage capacity.It is also portable and inexpensive so, it is specially used in mini computer and mainframe computer where a large volume of data needed to store.Dust,Humidity and temperature causes tape reading error.

SD-card(Smart Digital card)

SD-card refers to the Smart Digital card.It is also a secondary storage device.It is the most portable device.It can store minimum amount of data permanentaly.It stores 1-512GB.It is used in palmheld and tablet computer.Data are aslo stoted in on or off form.

SSD(Solid State Disk)

SSD refers to the Solid State Disk.It is the newest generation storage device.It is used instead of harddisk in laptop computer and desktop computer.It is faster than hard disk.It acts as a flash drive.It stored data in semi conductor.

Others

SMPS refers to the Switch Mode Power Supply.It supply all the powers required for all parts of computer.It is one of the most important internal parts of computer needed for running a computer.

Motherboard

Motherboard is a internal most important part of a computer.All the hardwares are connected into computers through motherboard.It is also called PCB(Printed Circuit Board).Motherboard is a main circuit board of a computer which passes electrical signal to all parts of computer.Different slots and pods are available in motherboard to connect different devices. Electrical pathway on the motherboard through which data and signal are passed to all parts of computeris called Bus.


Software

Instructions

Instructions is the basic command given to the computer to perform some tasks.

Program

Program is the set of instructions given to the computer to perform some tasks.

Software

Software is the set of program that makes computer hardware functioning.It perform it's specific task. Softwares are categorized into four types.

System Software

System Software is the set of programs that controls, Manages and operates all the activities of computer.It also allows to develop other programs. There are four parts of system software

OSS (Operating System Software)

OSS refers to the Operating System Software.It controls, manages and operates the overall activities of computer.It executes other applications.It acts as a interface between computer hardware and users. Some examples of OSS are: Windows 7. windows 8, windows 10, android, ios, linux and so on. There are two types of OSS

Single user and single tasking OSS

The operating system which allows only one users at a time.It performs only one task at a time. For eg:MS DOS

Multi user and multi tasking OSS

he operating system that allows more than one user at a time.It performs varieties or multiples tasks ata a time. For eg:Linux, Windows servers, Windows NT, Novel netware, and so on.

Function of OSS
  • Input/Output device management.
  • Processing management.
  • File management
  • Memory management.
  • Provides user interface
Provides user interface

User interface is the environment provides by OSS by which users interact with the computer. There are two types of user interface.

GUI(Graphical_User_Interface)

GUI refers to the Graphical Users Interface.It is very easier to work in GUI.It provides the Graphical working environment to the users.Such as:Icons, Buttons, Dialogue box, Text box, List box and so many others which we can see on our today's digital computers.

CUI(Command_User_Interface) or CLI(Command_Line_Interface)

CUI refers to the Command User Interface and CLI refers to the Command Line Interface.It doesnot provides the graphical working environment to the users. Users should type the command in order to get results.

Utility Software

Utility softwares is the set of programs that checks the system errors, repairs them and keeps personal computer healthy. For eg:Antivirus, Scan disk, Disk defragmanter, Disk clean up and so on that makes pc works faster

Device Driver

Device Driver is the set of programs that detects the new hardware component and makes functioning. For eg:Printer driver, Scanner driver, Keyboard driver, etc.

Language Processsor

Language processor is the program that converts the program written low level language and high level language into machine level language(Machine code). For eg:Assembler, Compiler and Interpreter.

Assembler

Assembler is a language translator that translate the programs written in Assembly language into Machine code that is 0 and 1.

Interpreter

Interpreter is a language processor that converts the program written in high level language into Machine code.It converts the one statement of a program at a time.

Compiler

Compiler is a language processor that converts the program written in High level language into machine code.It converts the whole statement of a program at a time.

Application Software

Application software is the set of programs that is designed to perform the specific task.It acts as a interface between user and operating system software.For eg: Ms-Excel, Ms-Powerpoint, Adobe photoshop, Medi-pro, Fact, Tally, etc. There are two types of Application software

Packaged Software

Packaged software is the set of programs that is designed by another group of company to meets the requirenents of a user. For eg: Ms-Excel, Ms-Powerpoint, Adobe photoshop, etc.

Customized or Tailored Software.

Customized or Tailored Software is the set of programs that is made on the demand of users. Specially, customized software is used for account management, Salary management, Inventory management, etc. For eg: Medi-pro, Fact, Tally, etc.

Programming language

Programming language is a computer language that is used to develop programs. For eg: COBOL(Common Business Oriented Language), FORTRAN(Formula Translator), QBASIC(Quick Beginners All Purpose Symbolic Instruction Code), Java, C, C++, PHP(Personal Home Page), Pearl, Python, Ruby, Milk, Anguler, PROLOG(Programmation en Logique), etc. There are three types of programming language

Machine Level Language

Machine level language is a computer language in which all the programs are written in the form of 0 and 1.It is machine dependent language and difficult to write the programs.

Assembly language.

Assembly language is a programming language in which programs are written by using alpha numeric word called Mneumonics.Programs are written like in commands:Add, Sub, Mpy, Cmp, etc.

High Level language.

High level language is a programming language in which programs are written by using English Words.It is easier to write the programs and it is a Machine dependent language. For eg:QBASIC, FORTRAN, C, C++, Java, Pearl, Python, etc.

Open source software

open source software is the type of software that allows to use, modify the source code and develop another software without permission of owner. For eg: PHP, Mysql, Linux, etc.

Freeware

The software which is freely available to use without paying any cost are called freeware. For eg: Browsers, Games available in Play store or App store and so many other application which can be use freely.

Web application

Web application are the server based application that needs internet connection to run. Information are stored in server computer which are provided to the clients. For eg: Facebook, Youtube, All the web browsers, Online service providing app, online games like: COC, Free Fire, PUBG, Call of duty, and so on.

Desktop application

Desktop application are not based on servers so that we can we use them without accessing of internet.In other word, those application which are designed to used offline and doesnot required internet service to run is called Desktop application.They can be use directly after installation on the computer. For eg:Ms office, VLC media player, Windows media player, and so many other offline application used in desktop or laptop computers.

Generation of computer

First generation of computer

First generation of computer were developed during 1946 AD to 1958 AD. They use vacuum tubes as a main electronic component. They are very slow in performance, and perform attacks in milliseconds. They were programmed in missing language. They will large in size and consume high electric power.

Second generation of computer

Second generation computer were developed during 1959 AD to 1964 AD. They used transistors as a main electronic component. They were faster and perform their tasks in a microsecond. They were programmed in Machine language, assembly and high level language.

Third generation of computer

Third generation computer were developed during 1965 AD to 1974 AD. They use IC (Integrated Circuit as a main electronic component. They were more faster than previous computer and perform their tasks in nanosecond. They use operating system, and could run different programs.

Fourth generation of computer

Fourth generation computer is developed in 1975 AD to onwards. They use VLSI(Very Large Scale Integration) and LSI(large Scale Integration) as a main electronic component. Their processing speed is increased and measured in pico second. They are more versatile and diligence.

Fifth generation of computer

Fifth generation computer is developed in present and will be in future. They use ULSI(Ultra Large Scale Integration) or biochips as a main electronic component. They have AI(Artificial Intelligence). They can think themselves, use intelligent programs, understand natural language, etc. They will have multi processors and parallel processing capability.


Hackers and Hacking



Introduction to Hacking

Hacking has been a part of computing for almost five decades and it is a very broad discipline, which covers a wide range of topics. The first known event of hacking had taken place in 1960 at MIT and at the same time, the term "Hacker" was originated. Hacking is the act of finding the possible entry points that exist in a computer system or a computer network and finally entering into them. Hacking is usually done to gain unauthorized access to a computer system or a computer network, either to harm the systems or to steal sensitive information available on the computer.Hacking is usually legal as long as it is being done to find weaknesses in a computer or network system for testing purpose. This sort of hacking is what we call Ethical Hacking.

Introduction to Hacker.

A computer expert who does the act of hacking is called a "Hacker". Hackers are those who seek knowledge, to understand how systems operate, how they are designed, and then attempt to play with these systems.

Types of hacking

We can segregate hacking into different categories, based on what is being hacked. Here is a set of examples

Website Hacking

Network Hacking

Hacking a network means gathering information about a network by using tools like Telnet, NS lookup, Ping, Tracert, Netstat, etc. with the intent to harm the network system and hamper its operation.

Email Hacking

It includes getting unauthorized access on an Email account and using it without taking the consent of its owner.

Ethical Hacking

Ethical hacking involves finding weaknesses in a computer or network system for testing purpose and finally getting them fixed.

Password Hacking

This is the process of recovering secret passwords from data that has been stored in or transmitted by a computer system.

Computer Hacking

This is the process of stealing computer ID and password by applying hacking methods and getting unauthorized access to a computer system.

Advantages of Hacking

Hacking is quite useful in the following scenarios

  • To recover lost information, especially in case you lost your password.
  • To perform penetration testing to strengthen computer and network security.
  • To put adequate preventative measures in place to prevent security breaches.
  • To have a computer system that prevents malicious hackers from gaining access.

Disadvantages of Hacking

Hacking is quite dangerous if it is done with harmful intent. It can cause

  • Massive security breach.
  • Unauthorized system access on private information.
  • Privacy violation.
  • Hampering system operation.
  • Denial of service attacks.
  • Malicious attack on the system.

Purpose of Hacking

There could be various positive and negative intentions behind performing hacking activities. Here is a list of some probable reasons why people indulge in hacking activities

  • Just for fun.
  • Show-off.
  • Steal important information.
  • Damaging the system.
  • Hampering privacy.
  • Money extortion.
  • System security testing.
  • To break policy compliance.
