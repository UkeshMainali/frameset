***Computer Security***

Computer Security

The protection given to the computer hardware and software from being lost, damage, theft, disaster and other intentional harm is called computer security.

Types of Computer Security

  1. Hardware security
    2. The protection given to the computer hardware from being lost, damage, theft and other intentional harms is called hardware security.

    Hardware security measures

    1. Regular maintenance.
      2. Hardware component is an important part of a computer. We have to check all the hardware parts time to time. If there is any error., we should repair them.
    2. Free from dust.
      3. Dust particles are the major causes of the failure of computer hardware that can damage different parts. We should clean the hardware parts time to time to secure the computer system.
    3. Insurance.
      4. Insurance is an important for the security of property. Computer and information stored in computer are also the property. So that we should insure the computer, we can get the insurance money for any lost and damage.
    4. protection from fire.
      5. Fire is one of the major destructive things.We should regularly check the wires and connector to avoid fire by minimizing short circuit.We should not play with fire in computer room because slight carelessnes of leads to danger. So, we should protect our hardware from fire otherwise our property will be loose and our self life also.
    5. Protection from theft.
      6. Theft is a kind of social evil so that we should protect the computer from theft. We should lock our main gate, windows, doors of home/building strongly. We can use the CCTC camera to observe environment of home.
    6. Air condition system.
      7. Computer automatically creates heat, over heat can damage the computer. So that computer room should be ventilated and fresh air should be flowed in a room. We should maintain the temperature of computer room 20 to 24 degree Celsius.
    7. Use of power protection devices.
      8. Some power protection devices can be used to protect the electronic devices such as computer, radio, TV, mobile etc. from the fluctuation of electricity. They are

      Power protection device

      1. UPS (Uninterruptible Power Supply)
        2. UPS is a power protection device. It has battery backup power. It protects the computer system from power failure and fluctuation of electricity and provides constant voltage to the computer.
      2. Volt Guard
        3. Volt Guard is a power protection device that controls the fluctuation (High and low voltage) of electric voltage and provides constant voltage to the computer system. Computer requires 220 volt to 240 volt of constant voltage.
      3. Spike Guard
        4. Spike means sudden over voltage of electricity. Spike guard protects the device from the sudden over voltage of current such as lightning.
      4. Voltage stabilizer
        5. Voltage stabilizer is common power protection device that protects the computer from frequent power failure of voltage fluctuation.
  2. Data and software security
    3. The protection given to the software from being loss, damage and other intentional harm is called software security. Software as well as data security is also major part because they have important role in any organization.

    Software security measures

    1. Password protection.
      2. Password is a secret word or alphanumeric word that protects the data, information and software from the unauthorized access of computer system. Password policy is available in a computer system.
    2. Biometric.
      3. Biometric is a technology that uses finger print, facial recognition, retina recognition, voice, digital signature for the security of data and information in computers system instead of password. It is more secure than the password policy.
    3. Backup system.
      4. Backup is the process of copying data and programs into another location or creating duplicate copy of its in a secured place. Backup system is important to save the data and programs from the accidental or intentional harm. We can recover data and programs if they are lost.
    4. USB port block.
      5. Hard lock key and USB port block is a data security mechanism. Some specific programs are used to lock the Hard Disk Drive and USB port. It protects the data from the unauthorized access of computer.
    5. Defragmentation.
      6. Disk Defragmentation is a software security measure that defrags the fragmented files of hard disk drive. It help to increase the performance of computer system. Disk Defragmenter is used to defrag the files on the disk.
    6. Scan disk.
    7. Antivirus.
      8. Antivirus is a Utility software that detects and removes the virus from the computer and keeps PC healthy like medicine.

Computer Virus

Computer virus is a program that replicates into the computer and destroys the normal functioning of a computer.

Purpose of creating computer virus.

  • To earn money.
  • To stop software piracy.
  • To steal data.
  • To entertain by displaying interesting message.

How the virus spread?

  • By using infected portable storage device like pen drive.
  • By opening virus infected email and email attachment.
  • By using pirated software.

Precautions for computer virus.

  • Always use antivirus software in computer.
  • Don’t use infected portable storage device like pen drive.
  • Scan the email and email attachment before opening it.
  • Don’t use pirated software.

Symptoms of computer virus.

  • Programs take more time to load.
  • Computer becomes hang frequently.
  • Unwanted message or picture appears on the screen.
  • Programs open automatically.

Effects of computer virus.

  • Replication of files.
  • It destroys data, information and files.
  • Unwanted message or pictures appears on the screen.
  • Reduces the performance of computer.
  • It loads and runs unwanted programs.

Types of computer virus.

Computer viruses are divided in different categories according to their nature and infection, they are…
  1. Boot sector virus
    2. Boot sector virus is a harmful program that infects the boot sector of floppy disk or MBR (Master Boot Record) of Hard Disk. EX: C-Brain, Stone, Disk Killer etc.
  2. File infector virus/parasitic virus
    3. File infector virus/parasitic virus infects the executable file of a program (files having .exe, .com, .bin, .sys extension) in a computer. Ex: Ambulance, Die Heart etc.
  3. Multipartite virus
    4. Multipartite virus is a combination of boot sector virus and file infector virus. It destroys the MBR of hard disk as well as executable files of programs in computer.
  4. Macro virus
    5. Macro virus infects the document files created in Microsoft office programs such as MS-Word, MS-Excel, MS-Power Point etc. Ex: Melissa, Love letter, etc.
  5. BIOS virus
    6. BIOS virus is a ransomware that affect the BIOS program (Firmware) of a ROM. It creates warning maessage that says the computer is infected by a virus. These messages leads to open a web browser and loading of page for fake antivirus software.
  6. Script virus
    7. Script virus is created in scripting language such as JavaScript, VBScript etc. It destroys the web page or files created in scripting language such as HTML, JavaScript. Ex: ‘HTML.Redlof.A’, ‘ILOVEYOU’
  7. Message carrying virus
    8. Message carrying virus does not infect the programs and files. It displays the unwanted picture or message on the screen and irritates the computer user.
  8. Computer worm
    9. Computer worm is a malicious program that does not infect other files. It replicates itself or creates multiple copies itself. Mainly, it blocks the internet connection and increases the network traffic.
  9. Trojan horse
    10. Trojan horse is a malicious program that appears as a game, antivirus etc. when it is installed into the computer, it helps to the hacker to bypass the sensitive information such as banking information, password or personal identity to the hacker.
  10. Polymorphic virus
    11. Polymorphic virus is a complex file infector that changes/modifies its appearance in every infection. It encrypts its code to secure from the antivirus.
  11. Stealth virus
    12. Stealth virus is a malware that uses various mechanisms to avoid detection by antivirus. It causes unrecognized icon appears, affects the printing devices, computer starts automatically, a system may crash without reason.
  12. Root kit
    13. Root kit is a set of software tools that enables an unauthorized user to gain of a computer system without being detected.
  13. Logic bomb
    14. Logic bomb is a piece of code intentionally inserted into a software that triggers by corrupting data, deleting files and cleaning hard drives when the specified conditions are met.
